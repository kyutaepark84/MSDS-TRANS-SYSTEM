# MSDS-TRANS-SYSTEM# MSDS-TRANS-SYSTEM

MSDS(물질안전보건자료) PDF를 입력하면, 고정된 PPT 템플릿에 맞춰
**현장경고표지**(GHS 라벨)와 **관리요령**(취급/보호구/응급조치 요약) 슬라이드
2종을 자동으로 PPTX 파일로 생성하는 도구입니다.

산업안전보건법 표준 16개 항목(KOSHA 서식) MSDS를 가정하며, 원본 문구를
규칙 기반으로 발췌·정리해 초안을 만들어 줍니다.

## 설치

```bash
pip install -r msds_ppt_generator/requirements.txt
```

## 사용법

```bash
python -m msds_ppt_generator "MSDS파일.pdf" --seq 21 --out-dir ./output
```

옵션:

| 옵션 | 설명 |
| --- | --- |
| `--seq` | 파일명 앞에 붙는 사내 관리번호 (예: `21`). 생략하면 번호 없이 생성 |
| `--rev-date` | 파일명에 쓸 날짜(`YYYY.MM.DD`). 생략하면 오늘 날짜 사용 |
| `--out-dir` | 결과 저장 폴더 (기본값: 현재 폴더) |
| `--only` | `label`(현장경고표지만) / `handling`(관리요령만) / `both`(기본값) |

결과 파일명 예시: `21.현장경고표지_NC-T30R_rev.2026.07.22.pptx`,
`21.관리요령_NC-T30R_rev.2026.07.22.pptx`

## 동작 방식

1. `msds_ppt_generator/msds_parser.py` — MSDS PDF의 페이지 머리말(반복 문구)을
   제거하고, 1~16번 항목 제목을 기준으로 본문을 분리한 뒤, 각 항목에서 필요한
   필드(제품명, 성분표, H/P-code, 개인보호구, 응급조치 등)를 정규식 기반으로
   추출합니다.
2. `msds_ppt_generator/ghs.py` — 추출된 H-code 목록으로부터 필요한 GHS
   그림문자(9종)를 표준 매핑표를 통해 계산합니다.
3. `msds_ppt_generator/ppt_builder.py` — `templates/label_template.pptx`,
   `templates/handling_template.pptx` 의 도형/표 서식은 그대로 두고 텍스트와
   그림문자 이미지만 교체해 새 PPTX를 생성합니다.

## 중요한 한계 (반드시 검토 후 배포하세요)

- **문서 서식 가정**: 파서는 이번에 제공된 MSDS와 같은 KOSHA 표준 16개 항목
  서식을 가정합니다. 항목 순서나 하위 항목 표기(가/나/다, 1)/2))가 다른 회사
  양식, 스캔 이미지 MSDS(OCR 필요) 에는 그대로 동작하지 않을 수 있습니다.
- **문구는 초안입니다**: "관리요령" 슬라이드의 문장은 MSDS 원문을 규칙 기반으로
  발췌·요약한 것으로, 사람이 다듬은 문장처럼 매끄럽지 않을 수 있고 위험성
  요약이 실제 작업 조건과 다를 수 있습니다. **인쇄/배포 전 안전보건 담당자
  검토가 필수**입니다.
- **그림문자 9종 중 6종은 간이 아이콘**: 템플릿에 실제로 포함돼 있던 느낌표/
  건강유해성/환경유해성 3종은 원본 이미지를 그대로 사용하지만, 인화성·산화성·
  고압가스·부식성·폭발성·급성독성(해골) 6종은 간단히 그려 넣은 대체
  아이콘입니다(`msds_ppt_generator/assets/pictograms/ghs0{1,2,3,4,5,6}_*.png`).
  실제 인쇄물에는 회사 표준 GHS 아이콘으로 교체하는 것을 권장합니다.
- **그림문자 우선순위 생략 규칙 미적용**: GHS 라벨 표시 기준에는 동일 유해성
  내에서 더 심각한 그림문자가 있으면 약한 그림문자를 생략하는 규칙이 있으나,
  이 도구는 H-code → 그림문자 매핑만 적용하고 그 생략 규칙은 적용하지
  않습니다.
- **표 행 높이는 고정**: 두 템플릿 모두 표 행 높이가 원본 예시(사람이 다듬은
  짧은 문장) 기준으로 고정돼 있습니다. 성분/유해성 문구가 유난히 많은
  MSDS는 표가 페이지 아래로 넘칠 수 있으니, 생성 후 반드시 PPT에서 한 번
  확인하세요(글자 크기를 살짝 줄이거나 문구를 다듬으면 해결됩니다).
- **동시 그림문자 개수**: 한 제품에 6종을 초과하는 그림문자가 필요한 경우
  우선순위(GHS01→GHS09) 상위 6개만 표시합니다.

## 폴더 구조

```
msds_ppt_generator/
  msds_parser.py       MSDS PDF -> 구조화 데이터
  ghs.py                H-code <-> 그림문자 매핑표
  ppt_builder.py        템플릿에 데이터 채워 PPTX 생성
  cli.py / __main__.py  커맨드라인 진입점
  templates/            고정 PPT 템플릿 2종(현장경고표지, 관리요령)
  assets/pictograms/    GHS 그림문자 9종 PNG
```
